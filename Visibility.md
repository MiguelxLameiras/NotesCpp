# Visibility

[Video being studied](https://www.youtube.com/watch?v=6OVQ8nh3KP0&list=PLlrATfBNZ98dudnM48yfGUldqGD0S4FFb&index=30)

- **Visibility** - refers to how and where parts of code, such as variables, methods, or classes, can be accessed from other parts of the program. For that it uses keywords **public**, **private**, **protected**.

    - **Public** - This keyword let's the specified part of the code be used inside and outside of the class.

    - **Private** - This keyword let's the specified part of the code be used only inside the class or as a **Friend**.

    - **Protected** - This keyword let's the specified part of the code be used inside the hierarchy of classes, such as base class and its respective sub-classes.
